# SexyLoot

## 📖 Descripción

SexyLoot es un addon para World of Warcraft 3.3.5a que reemplaza los mensajes tradicionales de obtención de objetos por notificaciones visuales modernas y totalmente personalizables.

Inspirado en los sistemas de "Loot Toast" de expansiones más recientes, SexyLoot muestra de forma elegante los objetos obtenidos, ganancias de oro, recetas aprendidas y otros eventos importantes mediante animaciones, sonidos y avisos visuales.

> [!NOTE]
> Compatible con World of Warcraft 3.3.5a.

> [!TIP]
> Dale a tu interfaz un sistema moderno de notificaciones para loot, oro y recompensas.

---

## ✨ Características

* ✔ Notificaciones visuales modernas.
* ✔ Seguimiento de objetos obtenidos.
* ✔ Seguimiento de oro ganado y perdido.
* ✔ Seguimiento de correo recibido.
* ✔ Seguimiento de intercambios.
* ✔ Avisos de recetas aprendidas.
* ✔ Sonidos personalizados.

* ✔ Totalmente configurable.
